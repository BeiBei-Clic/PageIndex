\chapter{使用文档}
\label{chap:usage}

\section{安装与配置}

\subsection{环境要求}

\begin{itemize}
    \item Python 3.10或更高版本
    \item 无需安装第三方依赖，仅使用Python标准库
\end{itemize}

\subsection{获取源码}

将\texttt{pathtracer}目录复制到项目中即可使用。

\section{命令行使用}

\subsection{基本用法}

\begin{lstlisting}[language=bash, caption=基本命令格式]
python -m pathtracer.cli [选项] <程序文件> [-- <程序参数>]
\end{lstlisting}

\subsection{命令行选项}

\begin{table}[htbp]
    \centering
    \caption{命令行选项说明}
    \label{tab:cli-options}
    \begin{tabular}{lll}
        \toprule
        \textbf{选项} & \textbf{说明} & \textbf{默认值} \\
        \midrule
        -f, --format & 报告格式（text/html/json） & text \\
        -o, --output & 输出文件路径 & stdout \\
        -q, --quiet & 静默模式，仅输出报告 & 否 \\
        -- & 分隔符，后接目标程序参数 & - \\
        \bottomrule
    \end{tabular}
\end{table}

\subsection{使用示例}

\textbf{示例1：生成文本格式报告}

\begin{lstlisting}[language=bash]
python -m pathtracer.cli agent_pageindex.py -- --query "test"
\end{lstlisting}

\textbf{示例2：生成HTML格式报告}

\begin{lstlisting}[language=bash]
python -m pathtracer.cli -f html -o report.html agent_pageindex.py -- --query "test"
\end{lstlisting}

\textbf{示例3：静默模式生成JSON报告}

\begin{lstlisting}[language=bash]
python -m pathtracer.cli -f json -o report.json -q agent_pageindex.py -- --query "test"
\end{lstlisting}

\section{API使用}

\subsection{基本追踪流程}

\begin{lstlisting}[language=Python, caption=API使用示例]
from pathtracer import PathTracer, CoverageReporter

# 创建追踪器并指定目标文件
tracer = PathTracer().trace_file("target.py")

# 开始追踪
with tracer:
    # 在此执行目标代码
    result = target_function(args)

# 生成覆盖率报告
reporter = CoverageReporter()
report = reporter.analyze("target.py", tracer)

# 输出报告
print(reporter.format_report(report))

# 保存HTML报告
with open("report.html", "w") as f:
    f.write(reporter.format_report_html(report))
\end{lstlisting}

\subsection{获取函数调用树}

\begin{lstlisting}[language=Python, caption=获取函数调用树]
# 获取追踪到的函数调用
calls = tracer.get_function_calls()

def print_call_tree(call, depth=0):
    indent = "  " * depth
    args = ", ".join(f"{k}={v}" for k, v in call.arguments.items())
    ret = f" -> {call.return_value}" if call.return_value else ""
    print(f"{indent}{call.function_name}({args}){ret}")
    for child in call.children:
        print_call_tree(child, depth + 1)

for call in calls:
    print_call_tree(call)
\end{lstlisting}

\section{报告格式说明}

\subsection{文本格式}

文本格式报告包含以下信息：

\begin{enumerate}
    \item 总体覆盖率统计（总分支数、已执行分支数、覆盖率百分比）
    \item 按分支类型统计
    \item 已执行分支详情（行号、分支类型）
\end{enumerate}

\subsection{JSON格式}

JSON格式报告结构：

\begin{lstlisting}[language=json, caption=JSON报告结构]
{
  "coverage_percentage": 31.8,
  "total_branches": 22,
  "executed_branches": 7,
  "branches_by_type": {
    "if": 17, "for": 5
  },
  "file_reports": {
    "target.py": {
      "executed_lines": [1, 2, 3],
      "executed_branches": [...]
    }
  }
}
\end{lstlisting}

\subsection{HTML格式}

HTML格式报告特点：

\begin{itemize}
    \item 响应式布局，支持各种屏幕尺寸
    \item 语法高亮的源代码显示
    \item 已执行/未执行代码行视觉区分
    \item 函数调用树可视化
\end{itemize}

\section{本章小结}

本章详细介绍了PathTracer工具的使用方法，包括命令行接口和Python API两种使用方式。工具支持三种报告输出格式，能够满足不同场景的需求。通过简单的配置即可对任意Python程序进行执行路径追踪和覆盖率分析。
