  <name>Task 1: Add FunctionCall model and tracing support</name>
  <files>pathtracer/models.py, pathtracer/tracer.py</files>
  <action>
    1. In models.py, add FunctionCall dataclass:
    ```python
    @dataclass
    class FunctionCall:
        """Records a function call during execution."""
        function_name: str
        file_path: str
        line_no: int
        call_depth: int
        arguments: Dict[str, str] = field(default_factory=dict)
        return_value: Optional[str] = None
        children: List['FunctionCall'] = field(default_factory=list)
    ```

    2. In tracer.py, extend PathTracer:
       - Add `function_calls: List[FunctionCall]` attribute
       - Add `_call_stack: List[FunctionCall]` attribute for tracking depth
       - In _trace_callback, handle 'call' events:
         - Extract function name from frame.f_code.co_name
         - Extract arguments from frame.f_locals (convert to string repr)
         - Create FunctionCall with current depth (len of _call_stack)
         - Append to parent's children if stack not empty, else to function_calls
         - Push to _call_stack
       - Handle 'return' events:
         - Pop from _call_stack
         - Set return_value using repr(arg) if arg is not None
       - Add get_function_calls() method returning the call list

    3. Update ExecutionPath in models.py to include:
       - `function_calls: List[FunctionCall] = field(default_factory=list)`
  </action>

  </verify>
