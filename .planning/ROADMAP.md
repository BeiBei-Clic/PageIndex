# Project Roadmap: PageIndex

## Project Overview
PageIndex is a vectorless, reasoning-based RAG system that builds hierarchical tree indices from long documents and uses LLMs for agentic, context-aware retrieval.

## Current State

- Search and retrieval capabilities working
