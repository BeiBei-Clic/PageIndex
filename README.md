# PageIndex

项目主代码保留在仓库根目录，课程报告、论文模板、学习 slides 和相关产物已统一收拢到 [`course-report/`](./course-report/)。

## 目录说明

- `pageindex/`：文档索引与检索核心实现
- `pathtracer/`：路径分析与覆盖率工具
- `tests/`：测试与实验脚本
- `course-report/`：课程报告源码、PDF、slides、模板素材

## 常用入口

- 运行说明：[`RUN.md`](./RUN.md)
- 编译课程报告：`make -C course-report pdf`
- 课程报告说明：[`course-report/README.md`](./course-report/README.md)
- 报告 PDF：[`course-report/main.pdf`](./course-report/main.pdf)
- Slides PDF：[`course-report/slides/main.pdf`](./course-report/slides/main.pdf)
